# Jeremymreed Tap

## How do I install these formulae?

`brew install jeremymreed/tap/<formula>`

Or `brew tap jeremymreed/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jeremymreed/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
